# Le Médaillier

Application web de gestion de collection numismatique — 100 % hors-ligne, installable sur mobile (PWA).

**© 2026 Kévin François Cagnat — Tous droits réservés.**

## À propos

Le Médaillier permet de cataloguer une collection de pièces de monnaie : fiches détaillées avec photos, grades (échelle française), estimation de valeur et de valeur métal, suivi de séries (avec variations par atelier et par tranche), carte du monde, wishlist, registre des ventes, fiscalité de cession et bien plus. Les données restent stockées localement sur l'appareil de l'utilisateur (IndexedDB) ; le fichier hébergé ne contient aucune donnée personnelle.
